Arrays

This folder contains fundamental array problems that build core problem-solving skills. 
Repository also includes solution of the problems given on the other platforms like GeeksForGeeks and Leetcode.

Problems Included
- Two Sum
- Kadane's Algorithm (Maximum Subarray Sum)
- Move Zeros
- Best Time to Buy and Sell Stock
- Missing Number
- Check if Array is Sorted and Rotated
- Rotate Array
